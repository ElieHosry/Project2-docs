Goal workbook templates

goal-workbook-templates.zip — the M1 + M2 goal workbooks for every category (20 .xlsx files — all 10 categories, one folder each).

Each workbook contains:

  • a Team Info sheet (team, project, category, milestone, evaluation date, instructor) — filled in when a team derives its copy;
  • a Goals sheet with the 6 milestone goals: G1–G3 ⭐ common goals adapted to the category, plus the category's 3 most representative pool picks (# = pool number), each with What the team must demonstrate and 2–5 measurable acceptance criteria;
  • an Owner column (one student per goal), a Status dropdown (Done / Partially Done / Missing / Untested) and an Instructor Feedback column for the milestone evaluation.

How they're used: after a proposal is approved, the team receives (or the Proposal Assessor generates) a copy tailored to their project — these templates are the per-category starting points. The footer rule applies to every goal: demonstrated live, integrated into the team's main build, on the mobile target (see Appendix F in the Appendices).

Proposal Assessor skill (Claude / AI agents)

Project-Proposal-Assessor.skill — the installable assessment skill: three-tier verdicts against the 10 categories, web-verified references, requirements tables, suggested goals and generated goal workbooks.

Install by dropping the file into a Claude desktop (Cowork) chat and clicking Save skill; it also works in other SKILL.md-compatible agents (unzip into their skills directory). Full instructions, capabilities and maintenance notes: Using the Project Proposal Assessor.
